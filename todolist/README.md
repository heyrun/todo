# Todo List Application

A full-stack todo list application with Ruby backend and React frontend

## Features
- Add new todos
- Mark todos as completed
- Delete todos
- Real-time updates
- Responsive design
- Strikethrough completed items

## Tech Stack
- **Frontend**: React.js
- **Backend**: Ruby on Rails
- **Database**: SQLite (development)
- **Styling**: CSS Modules
- **Icons**: React Icons

## Installation

### Backend Setup
```bash
cd backend
bundle install
rails db:setup
rails server
```

### Frontend Setup
```bash
cd frontend
npm install
npm start
```

## Usage
1. Add todos using the input field
2. Check/uncheck todos to mark completion
3. Click the trash icon to delete items
4. Todos persist between sessions
